## What Is the Image Library?

The image library is a shared collection of all images uploaded by blog authors. Instead of uploading the same image twice, you can browse the library and insert an existing image into any post. This keeps things organized and avoids duplicates.

## Opening the Image Gallery

1. Open the blog editor (create or edit a post).
2. Below the body text area, click **Browse Images**.
3. A gallery modal opens showing all uploaded images as a scrollable grid.

## Searching for an Image

At the top of the gallery, there's a search field. Type part of an image title and the results update automatically. If no images match your search, you'll see a "No images match your search" message.

Clear the search field to see all images again.

## Inserting an Image from the Gallery

Each image card in the gallery shows a thumbnail, the image title, and the alt text. To insert an image into your post:

1. Find the image you want.
2. Click the **Insert** button on the image card.

The image tag (e.g., `{{image:42}}`) is added to the end of your post body. Move it to wherever you'd like it to appear.

## Image Cleanup

You don't need to worry about cleaning up unused images yourself. The system automatically tracks which images are used in posts. If an image isn't referenced by any post for an extended period, it's cleaned up automatically. If you insert an image into a post and later remove it, the system notices and handles things from there.
