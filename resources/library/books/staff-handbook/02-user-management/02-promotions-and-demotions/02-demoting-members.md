## Overview

Demotions move a user down one membership level. They're less common than promotions but sometimes necessary. A user can't be demoted below **Traveler** -- if more serious action is needed, use the [[books/staff-handbook/user-management/the-brig/putting-users-in-the-brig|Brig]] instead.

## Who Can Do This

- Admins
- Officers (any department)
- Crew Members in the **Quartermaster** department

## How to Demote a Member

1. Navigate to the user's **profile page**
2. Click the **three-dot menu** on their profile card
3. Select **Demote to {previous level}** (e.g., "Demote to Traveler")
4. Confirm in the modal that appears

The demote option only appears when:
- You have permission to manage users
- The user is above Traveler level
- The user is not in the Brig

## What Happens

When you demote a user:
- Their membership level drops by one
- Their **Minecraft rank** syncs to match the new level
- Their **Discord roles** update accordingly
- The action is recorded in the activity log
- **No notification is sent** to the user -- you may want to reach out to them directly to explain

## Important Notes

- You cannot demote someone below Traveler. Stowaways and Drifters are entry levels that users only pass through once.
- Demotions don't update the user's `promoted_at` date -- that only changes on promotions.
- If a user is in the Brig, the demote option is hidden. Handle the Brig situation first.
